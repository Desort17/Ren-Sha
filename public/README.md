This folder is for static assets like images and audio. 

**MANDATORY STEP:**
1. Use the "Upload" button in the file explorer to add your files.
2. Place them EXACTLY in this `public` folder.
3. Ensure they are named:
   - `Sha.png`
   - `Ren.png`
   - `romantic.mp3`
   - `lylia.png`

Vite will automatically serve these files from the root of your website.
